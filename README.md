# 💻 Laptop Price Predictor

This is an AI-based web app that predicts laptop prices based on specifications.

## 🚀 Features
- ML-based prediction
- Beautiful UI (Streamlit)
- Real-time price estimation

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- XGBoost
